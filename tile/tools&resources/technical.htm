Technical Information

The TILE services are designed to be as hardware independent as possible, making extensive use of Java and XML. The web services are implemented as Java servlets. The learning modules, metadata, preferences, and session information are all recorded using XML. The learning modules, metadata, preferences are implemented using open IMS specifications (Content Packaging, Meta-data, and Learner Information Package Accessibility respectively).

The TILE website and transformations will function in any web browser on any platform (Linux, Mac OS, and Windows), but we do make extensive use of Cascading Style Sheets. Mozilla (http://www.mozilla.org) offers the best support for style sheets, and will provide the best experience.
