Authoring Tool

The TILE Authoring Tool allows registered content authors to create content packages and publish them in the TILE repository. The Authoring Tool user interface walks the user through the process of uploading and assembling their learning content. It allows authors to define the learning objectives they wish their students to achieve, and to assign resources students will use to achieve these objectives.

An important feature offered by the TILE Authoring Tool is the ability for the author to identify alternative resources for achieving learning objectives. This allows authors to support students with varying learning styles and accessibility needs by providing content in different modalities and forms, such as providing text transcripts of audio files, or a simulation as an alternative to a theoretical explanation.
